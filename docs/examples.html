examples.darias_teachingmode_cartesianpace module¶

Example 1. Set the robot in teaching mode 2. Record the trajectory (in RL the left arm will just drop down, on the real robot the user can move it in gravity compensation) 3. The arm will be brought to its initial position 4. The recorded trajectory will be imitated.

Everything will happen in the cartesian space.

examples.darias_teachingmode_conditional_recording module¶

Example 1. Set the robot in teaching mode 2.1. The recording starts only when the user start moving the robot 2.2. Record the trajectory (in RL the left arm will just drop down, on the real robot the user can move it in gravity compensation) 2.3 The recording stops when the user is not moving the arm anymore 3. The arm will be brought to its initial position 4. The recorded trajectory will be imitated.

Everything will happen in the joint space.

Example 1. Set the robot in teaching mode 2. Record the trajectory (in RL the left arm will just drop down, on the real robot the user can move it in gravity compensation) 3. The arm will be brought to its initial position 4. The recorded trajectory will be imitated.

Everything will happen in the joint space.
