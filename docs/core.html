core.darias_interface module¶

This module contains the interface for Darias. Darias is viewed as an object with a set of joints and two end-effectors.

The end-effector (left and right) expose their positions in 3D coordinates (with relative orientations). The joints can be divided in two logical groups (left and right), and they expose the angle in radiants. In future the joints will be divided in four groups (left-arm; left-hand; right-arm; right-hand).

Darias interfaces exposes also two methods, one for control and one for kinesthetic teaching.

conditional_record(callback_start, callback_end, max_duration=60.0)[source]¶

This is a conditional recording. the recording will start as soon as callback_start will return true.

Parameters:
  • callback_start – Function which receives a goal as input, and produce a decision. When the decision is true, then the recording starts
  • callback_end – Function which receives a goal as input, and produce a decision. When the decision is true, then the record will stop, and this method will return.
  • max_duration – the overall process cannot exceed the max_duration, expressed in seconds.
